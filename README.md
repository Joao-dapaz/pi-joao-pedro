# 🎵 Maestro - Conexões Musicais

O **Maestro** é uma aplicação web desenvolvida como projeto integrador do SENAC TECH, com o objetivo de conectar alunos, professores e escolas em um ambiente digital de ensino musical.

Inspirado em plataformas como o Google Classroom, o sistema permite o gerenciamento de turmas, compartilhamento de materiais e interação entre os usuários de forma simples e organizada.

## 🚀 Funcionalidades

* Autenticação de usuários (Aluno, Professor e Escola)
* Gerenciamento de turmas
* Vinculação de professores e alunos às turmas
* Publicação de materiais pelos professores
* Visualização e download de materiais pelos alunos
* Organização das turmas por escola

## ▶️ Como Executar o Projeto

1. Clone o repositório:

```bash
https://github.com/Joao-dapaz/pi-joao-pedro.git
```

2. Acesse a pasta:

```bash
cd pi-joao-pedro
```

3. Crie e ative o ambiente virtual:

```bash
python -m venv venv
venv\Scripts\activate
```

4. Instale as dependências:

```bash
pip install flask
```

5. Execute o sistema:

```bash
python app.py
```

6. Acesse no navegador:

```bash
http://localhost:5000
```
